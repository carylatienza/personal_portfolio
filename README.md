# Caryl Atienza — Personal Portfolio

A personal portfolio website showcasing my journey as an aspiring Full-Stack Developer with a background in Graphic Design.

## 🔗 Live Demo
[View Portfolio](https://carylatienza.github.io/personal_portfolio)

## Built With
- HTML5
- CSS3
- JavaScript
- [Font Awesome](https://fontawesome.com/) — icons

## Features
- Hero section with personal introduction
- About Me section with Skills, Experience, and Education tabs
- Services section
- Portfolio/Work showcase with project previews
- Contact section with social media links

## Known Limitations
- Contact form is not yet functional
- "See More" button is not yet implemented
- Mobile responsiveness is still in progress

## Getting Started
To run this locally, simply clone the repo and open `index.html` in your browser:

```bash
git clone https://github.com/carylatienza/personal_portfolio.git
cd personal_portfolio
open index.html
```
